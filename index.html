Recent Projects

Project 1

To-Do List

This to-do list web application, titled "Just Do It," allows users to add, complete, and delete tasks in an organized manner. The minimalist design features a dark-themed background with a gradient effect, giving the app a modern look. Users can input tasks through the form, which are then displayed below with options to mark tasks as complete or delete them. The application also displays the current date and time, enhancing the user experience by keeping track of daily activities. The overall project is a simple yet effective way to manage tasks, showcasing my skills in HTML, CSS, and JavaScript

Project 2

Tea Kadai 3D Model

This 3D model represents a small, traditional "Tea Kadai" (tea shop), a common sight in local neighborhoods. Created using Blender, the model showcases essential elements like seating, potted plants, and a large teacup on the roof, adding a touch of charm and cultural relevance. I focused on a low-poly style for a clean and minimalistic look, aiming to evoke a familiar and inviting atmosphere.
